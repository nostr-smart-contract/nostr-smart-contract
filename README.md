# Nostr Smart Contract :coin:

Mini demo/PoC for smart contracts using Nostr over Bitcoin timechain

## About 

This is a simple PoC for smart contracts using Nostr over Bitcoin timechain.

## How to run

- Clone the repo
- Run `npm install`
- Run `npm run dev`
- Visit `http://localhost:3000`


## Libraries used

- Bitcoinjs-lib
- NDK
